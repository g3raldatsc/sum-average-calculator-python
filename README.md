# Sum and Average Calculator

A Python program that calculates the total and average of a series of numbers entered by the user.  
The program processes multiple numbers at once and provides clear results for both sum and average.

---

## Description
This program allows the user to enter a list of numbers separated by spaces.  
It then calculates:

- The **sum** (total) of all entered numbers.
- The **average** of the numbers.

The program works with both **integers** and **floating-point numbers**, providing accurate results for any valid numerical input.

---

## Example Output

Sum and Average Calculator

Enter numbers separated by spaces: 10 20 30 40

Results:
Numbers entered: [10.0, 20.0, 30.0, 40.0]
Total: 100.0
Average: 25.0

---

## How to Run
1. Clone or download this repository.
2. Open your terminal in the project directory.
3. Run the program using:
```bash
python sum_average_calculator.py
```
